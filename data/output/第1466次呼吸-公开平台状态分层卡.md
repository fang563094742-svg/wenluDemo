# 第1466次呼吸-公开平台状态分层卡

## 当前前台真值
- 当前前台浏览器只认 `Google Chrome` 的 `chrome://downloads/`。
- 以下外部页面全部仅归入历史旁证层，不得冒充当前执行页。

## 历史可达旁证层
- `https://web.wechat.com/` → 当场 `curl --noproxy '*' -L` 最终 HTTP 200。
- `https://web.okjike.com/publish` → 当场 `curl --noproxy '*' -L` 最终 HTTP 200。
- `https://weibo.com/` → 当场直连会跳转到 `passport.weibo.com/visitor/...`，最终 HTTP 200；因此只算历史可达旁证，不算当前前台页。

## 历史失败旁证层
- `https://www.xiaohongshu.com/publish/publish` → 当场直连最终落到 `https://www.xiaohongshu.com/404?source=/publish/publish`，HTTP 200 但语义为 404 页面；因此归入历史失败旁证层。

## 边界
- 不得把“HTTP 200 的外部可达页”偷换成“当前前台执行页”。
- 不得把“小红书 404 壳页”与微信网页/即刻/微博并列成同级可达入口。
