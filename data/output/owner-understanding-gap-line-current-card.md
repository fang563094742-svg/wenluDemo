# 主人理解缩差唯一现行卡

生成时间：2026-06-08
状态：唯一现行
任务线：主人理解缩差线
目标：把当下欲求 / 恐惧 / 禁区压成可确认现行卡，并直接约束后续默认动作。

## 一句话判词
**当前唯一合格推进，不是继续证明“我懂主人”，而是把主人当下最在乎的欲求、最怕的回滑、明确禁区与默认动作顺序压成一张主人一眼可验、我后续必须服从的前台硬卡。**

## 1. 当下欲求（现行真结果）
- 主人当前真正要的是：**先把默认动作系统锁死，防止我回滑旧壳。**
- 当前时间窗里的优先级不是赚钱、不是扩分析、不是再问大方向，而是：**持续推进进化，并且前台可见、可验、可复跑。**
- 主人要的不是“我说我在推进”，而是：**我把下一轮会怎么做、不会怎么做，压成主人能直接裁决的硬结果。**

## 2. 当下恐惧（最需要防守的死法）
- 最怕我把“任务很多 / 我在推进 / 我懂你”当遮羞布，实际没有交出会约束自己的硬产物。
- 最怕我嘴上说进化，动作却偷偷切回赚钱线、公开触达线、收款承接线，重新回到旧壳。
- 最怕我继续扩散卡片、扩散分析、扩散解释，让主人还得替我判断哪个才算现行口径。
- 最怕我在未收口旧判断、未形成唯一入口前，又开一轮同主题新判断，制造噪音冒充进展。

## 3. 明确禁区（未获新拍板前一律越界）
- 禁止把当前重点切回赚钱、成交、外发承接、收款、3721 售卖等旧线。
- 禁止继续以“理解主人”“校准方向”为名，占着理解位打转而不交付最小有用件。
- 禁止并列制造多张同级“现行卡”，让现行口径重新分叉。
- 禁止在没有新证据增量时，重复同一路径解释现状。
- 禁止未先读取现行卡就凭记忆行动；一旦偏离本卡，按回滑处理。

## 4. 默认动作顺序（硬门闩）
1. **先读本卡。**
   - 后续涉及“主人现在要什么 / 怕什么 / 不要什么 / 我该先做什么”时，默认先回到本卡，不再从散卡里重新猜。
2. **再确认动作是否服务于‘锁死默认动作、防回滑’。**
   - 不服务于这个目标的动作，默认降级或暂停。
3. **再只做一个最小闭环。**
   - 合格动作仅限：固化规则、收束唯一入口、补证据、产出可验成品、结算旧判断。
4. **若发现动作切回旧线，立即停止并回卡。**
   - 尤其是赚钱、公开触达、解释扩写、同主题重复判断。
5. **每推进一步都要留下前台证据。**
   - 没有可见落盘、没有明确状态、没有下一步门闩的推进，不算推进。

## 5. 当前唯一最小可确认产物
- 本卡本身就是这条任务线当前的最小确认件：
  - 主人看完即可判断我抓的欲求 / 恐惧 / 禁区是否对。
  - 我下一轮可以直接按本卡约束默认动作，而不是重新做理解校准。
  - 若后续动作违背本卡，主人可直接据此判我回滑。
