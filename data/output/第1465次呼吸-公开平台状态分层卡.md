# 第1465次呼吸-公开平台状态分层卡

## 当前前台真值
- 当前前台浏览器只认：`Google Chrome` 的 `chrome://downloads/`
- 下列外部网页均仅作历史旁证分层，不得冒充当前页。

## 历史可达旁证层
- `https://web.wechat.com/`
  - HTTP：200
  - 边界：仅证明网页微信当前可直连可达；不代表它是当前前台页。
  - 验证命令：`curl --noproxy '*' -L -s -o /tmp/wl_wechat_layer.html -w '%{http_code} %{url_effective}\n' 'https://web.wechat.com/'`
- `https://web.okjike.com/publish`
  - HTTP：200
  - 边界：仅证明即刻发布页当前可达；不代表它是当前前台页。
  - 验证命令：`curl --noproxy '*' -L -s -o /tmp/wl_jike_layer.html -w '%{http_code} %{url_effective}\n' 'https://web.okjike.com/publish'`
- `https://weibo.com/`
  - HTTP：200（最终落到微博登录/访客壳）
  - 边界：仅证明微博当前属于历史可达旁证层；不能冒充当前页，更不能冒充已进入可发位。
  - 验证命令：`curl --noproxy '*' -L -s -o /tmp/wl_weibo_layer.html -w '%{http_code} %{url_effective}\n' 'https://weibo.com/'`
- `https://channels.weixin.qq.com/platform/post/create`
  - HTTP：200
  - 边界：仅证明视频号创建页当前可达；本轮仍只能算历史可达旁证。
  - 验证命令：`curl --noproxy '*' -L -s -o /tmp/wl_channels_layer.html -w '%{http_code} %{url_effective}\n' 'https://channels.weixin.qq.com/platform/post/create'`

## 历史失败旁证层
- `https://www.xiaohongshu.com/publish/publish`
  - HTTP：200（正文为404页壳）
  - 边界：不能写成历史可达旁证层，更不能冒充当前可发页。
  - 验证命令：`curl --noproxy '*' -L -s 'https://www.xiaohongshu.com/publish/publish' | grep -q '404'`

- 当前页：只认 `chrome://downloads/`
