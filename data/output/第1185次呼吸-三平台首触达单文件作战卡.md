# 第1185次呼吸-三平台首触达单文件作战卡

生成时间：2026-06-08 15:34 CST
目标：把你剪贴板这句压成一张可直接复制的单文件卡；只解决“别人一回你，我下一句发什么”。
验收：你只需要判断这张卡对你有没有用。

## 默认打法
- 先发平台钩子句。
- 对方一旦回“能做吗 / 想聊 / 感兴趣”，直接贴对应首回。
- 若对方不回预算，只继续拿时间、现有材料、交付形态；不要自己先展开解释。
- 若 10 分钟还没回，就不补第二段长文，换下一个目标。

## 平台钩子句
### 小红书
你这个我能接，给我需求和时间，我直接帮你收口。

### 即刻
这类活我能接，给我目标和截止时间，我直接给你可交付版。

### 微博/私信
这单我能做，你把需求、最晚时间、预算区间丢我，我直接回你怎么收口。

## 统一首回正文
我能接，先卡 3 个信息我就直接报：
1. 最晚什么时候要？
2. 你现在手上已有啥（文案 / 素材 / 账号 / 链接）？
3. 你更想要我交付成哪种：直接可发版 / 方案框架 / 我代你收口？
你直接私我这 3 项，我按小单最快给你收口。

## 对方追问时的最短补句
### 问价格
可以，先按小单试做，我看你这单信息量和时效，直接给你一口价。

### 问多久能给
你把上面 3 项丢我，我先按最快版给你排，急单我就按加急方式收口。

### 问能做到哪一步
我可以按你要的深度来：直接可发版、方案框架版，或者我直接帮你收口成成品。

## 换挡条件
- 对方连续两次只问空问题、不肯给 3 项信息：结束，不继续解释。
- 对方只想白嫖方案：只给最小边界，不进入深聊。
- 对方愿意给时间 + 材料：立刻进入报价或交付承接。

## 现状边界
- 这是一张“首触达承接卡”，不是发送留证。
- 还不算已进钱；只算把你下一句减少到可直接复制。
